# Sistema de Banco de Dados - Consultório Médico

Projeto desenvolvido com foco em modelagem de banco de dados relacional utilizando MySQL.

O sistema simula a estrutura de um consultório médico, permitindo o registro e organização de pacientes, médicos e consultas.

##  Estrutura do Banco de Dados

O banco é composto por tabelas relacionadas para representar:

-  Médicos
-  Pacientes
-  Consultas
-  Informações médicas

Foram aplicados conceitos de:

- Chaves primárias
- Chaves estrangeiras
- Relacionamentos entre tabelas
- Integridade referencial
- Normalização

##  Tecnologias Utilizadas

- MySQL
- SQL

##  Funcionalidades Implementadas

- Criação das tabelas do sistema
- Definição de relacionamentos entre entidades
- Inserção de dados para testes
- Consultas utilizando:
  - SELECT
  - WHERE
  - JOIN
  - ORDER BY

##  Objetivo do Projeto

Praticar modelagem de banco de dados, criação de estrutura relacional e execução de consultas SQL aplicando conceitos vistos em Análise e Desenvolvimento de Sistemas.

##  Estrutura do Repositório

- Script SQL para criação do banco
- Script para inserção de dados
- Consultas para extração de informações
